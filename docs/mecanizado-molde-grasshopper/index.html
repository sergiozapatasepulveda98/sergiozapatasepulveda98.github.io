Mecanizado Molde Grasshopper

Molde Grasshopper

El siguiente ejercicio consiste en la elaboración de un objeto “molde”, a partir extracción de material determinada por la repetición en patrones de un cuerpo sobre otro. Este cuerpo fue construido en Rhino, usando Grasshopper. Se tomó como figura base un prisma rectangular y se le extrajo material a partir de una esfera repetida en patrones de 8x8. A continuación se presenta el paso a paso.
